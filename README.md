# Cine.tv
Es una plataforma web desarrollada para ver películas y series de forma rápida y sencilla. Es un proyecto de aceso gratuito, se enfoca en una interfaz limpia, responsiva y con una experiencia de usuario inspirada en las grandes plataformas de contenido audiovisual.

🚀 Características principales
1- Diseño Responsive: Adaptable a diferentes tamaños de pantalla (Desktop y móvil).

2- Interfaz de Usuario: Inspirada en plataformas modernas de streaming (estilo oscuro, tipografía legible y botones de acción claros).

3- Navegación Intuitiva: Menú de categorías (Inicio, Películas, Series, Animación) para fácil acceso.

4- Detalles por Película: Sistema de enlaces para acceder a descripciones detalladas de cada contenido.

5- Buscador Integrado: Formulario funcional para filtrar contenido por nombre.

🛠️ Tecnologías utilizadas
- HTML5: Estructura semántica del contenido.

- CSS: Estilos personalizados, Flexbox/Grid para el diseño y efectos de hover/transiciones.
